# 楽譜と装置の準備  

手回し演奏モードで演奏するには、背面カメラで読み取る楽譜と楽譜を送る装置が必要です。

現状楽譜としては40mm/70mm幅のロール楽譜あるいはディスク楽譜が使え、これはアプリから普通紙に印刷することができます。

「楽譜の印刷」を参照

また楽譜を送る装置としては、市販の手回しオルゴール、手回し演奏玩具、回転台などがあります。
