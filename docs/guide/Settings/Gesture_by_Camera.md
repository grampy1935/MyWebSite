# カメラによるジェスチャー

「APP内課金」の「カメラを使う機能」を購入すると表示されます。

オンにすることにより、カメラによるジェスチャー演奏が有効になります。

また、オンの場合には「手回し」と「指揮者」のモードが切り替え可能となります。
